# Uber Ride Data Analysis Dashboard

## Project Overview

This project analyzes **Uber ride booking data** to understand ride patterns, revenue trends, customer behavior, and operational performance. The analysis helps identify key business insights that can improve **customer experience, operational efficiency, and revenue generation**.

The project includes an interactive dashboard that provides insights into bookings, revenue, rider behavior, vehicle performance, and location-based trends.

---

# Business Problem

Ride-hailing companies generate large volumes of data from bookings, payments, customer interactions, and driver activities. However, extracting actionable insights from this data is essential to improve decision-making.

The goal of this project is to analyze Uber ride data and answer the following business questions:

* What factors influence booking completion and ride cancellations?
* Which vehicle types generate the highest revenue?
* What locations and time slots experience the highest demand?
* How do customer behaviors differ between new riders and returning riders?
* What payment methods are most frequently used?

---

# Dashboard Overview

The dashboard is divided into multiple analytical pages to provide a comprehensive view of Uber’s ride operations.

---

# 1. Overview Page

This page provides a **high-level summary of business performance** using key metrics and trend analysis.

### Key Performance Indicators (KPIs)

* Completed Bookings
* Lost Bookings
* Total Revenue
* Total Distance Travelled
* Average Ride Distance

### Key Visualizations

* Monthly Booking Completed
* Monthly Revenue Analysis
* Quarterly Booking Completed
* Quarterly Revenue Analysis
* Revenue by Vehicle Type
* Top Pickup Locations by Booking Count
* Top Drop Locations by Booking Count

### Ratings Analysis

* Average Rider Rating
* Average Driver Rating

### Filters

* Vehicle Type Filter
* Date Filters
* Interactive Dashboard Controls

---

# 2. Vehicle Page

This page provides detailed insights into the performance of different vehicle categories.

### Metrics

* Total Booking Count by Vehicle
* Revenue Generated by Vehicle Type
* Contribution Percentage of Each Vehicle Category

This analysis helps determine which vehicle categories contribute the most to revenue and ride demand.

---

# 3. Revenue Page

This section analyzes revenue distribution across multiple dimensions.

### Revenue Breakdown

* Revenue by Customer
* Revenue by Vehicle Type
* Revenue by Payment Method

### Time-Based Analysis

* Monthly Revenue Trends
* Quarterly Revenue Trends

This helps identify which customers, vehicles, and payment methods generate the highest revenue.

---

# 4. Rider Page

This page focuses on customer behavior and ride cancellation patterns.

### Key Insights

* Ride Cancellations by Reason
* Payment Method Distribution
* Monthly Ride Trends
* Quarterly Ride Trends

### Rider Segmentation

* First-Time Riders
* Returning Riders
* Regular Riders

A detailed data table is included to allow deeper analysis of rider behavior.

---

# 5. Location Page

This section analyzes geographic and time-based ride demand.

### Key Metrics

* Monthly Total Distance Travelled
* Total Distance by Vehicle Type
* Peak Ride Time Slots
* High Demand Areas (Busy Locations)

This analysis helps identify high-demand areas and peak usage times.
